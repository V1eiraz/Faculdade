#  C - 1º Semestre: Engenharia de Computação

Este repositório armazena todos os projetos, exercícios e trabalhos desenvolvidos na disciplina de **[Nome da Disciplina, ex: Introdução à Programação ou Algoritmos e Estruturas de Dados I]** durante o primeiro semestre do curso de Engenharia de Computação.

O objetivo é documentar meu progresso, praticar os conceitos fundamentais da linguagem C e criar um portfólio dos meus primeiros passos na programação.

## 📚 Conteúdos Abordados

Os códigos aqui presentes exploram os seguintes tópicos fundamentais da linguagem C:

* Variáveis, Tipos de Dados e Operadores
* Entrada e Saída (Scanf e Printf)
* Estruturas Condicionais (if, else, switch)
* Estruturas de Repetição (while, for, do-while)
* Vetores (Arrays) e Matrizes
* Strings (Manipulação de Cadeias de Caracteres)
* Funções e Escopo de Variáveis
* Ponteiros
* Alocação Dinâmica de Memória (malloc, free)
* Structs (Estruturas de Dados)
* Manipulação de Arquivos (leitura e escrita)

## 🚀 Como Compilar e Executar

A maioria dos projetos pode ser compilada utilizando o GCC (GNU Compiler Collection).

```bash
# Navegue até a pasta do projeto
cd pasta-do-projeto

# Compile o arquivo .c
gcc -o executavel nome_do_arquivo.c

# Execute o programa
./executavel
```


Para compilação com mais flags (recomendado para depuração):
```bash
gcc -Wall -Werror -std=c99 -o executavel nome_do_arquivo.c
```
